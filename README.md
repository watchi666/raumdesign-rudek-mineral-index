# RaumDesign Rudek — Redesign

Eigenständiges statisches Redesign für RaumDesign Rudek in Düsseldorf.

## Lokal starten

```powershell
npx serve . -l 4173
```

Alternativ kann jeder einfache statische HTTP-Server verwendet werden.

## Struktur

- `index.html` — Startseite
- `impressum.html` — Impressum
- `datenschutz.html` — Datenschutz
- `404.html` — Fehlerseite
- `assets/css/style.css` — Designsystem und Responsive Styles
- `assets/js/main.js` — Menü, Reveals, Galerie und Mailto-Formular
- `assets/images/` — echte Projektbilder der vorhandenen Website
- `assets/fonts/` — lokal eingebundene Schriften
- `DESIGN.md` — Designsystem
- `REBUILD_PLAN.md` — Strategie und Farbentscheidung
- `COPY_CHANGES.md` — Textentscheidungen
- `QA_REPORT.md` — technische und visuelle QA
- `ACCESSIBILITY_AUDIT.md` — Accessibility-Prüfung

## Datenschutz

Die Website verwendet keine Cookies, Analysedienste, eingebetteten Karten oder extern geladenen Schriften. Das Formular bereitet ausschließlich eine E-Mail im lokalen E-Mail-Programm vor.

## Vor offizieller Veröffentlichung

Impressum, Datenschutz, Bildrechte und sämtliche Betriebsdaten müssen vom Betreiber rechtlich und fachlich bestätigt werden.
